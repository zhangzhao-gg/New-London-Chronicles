# PRD: 新伦敦计划 (New London Project)

## 产品概述
一个基于冰汽时代美学的多人共享专注工具。用户通过完成番茄钟专注，为同一座城市"新伦敦"做贡献。城市永久记录每个用户的贡献。

**核心一句话**: 专注工作，建造新伦敦，城市会记得每一个人

---

## 技术栈
- **前端**: Vue 3 + Vite
- **后端**: Node.js (Express)
- **数据库**: PostgreSQL
- **部署**: 网页端，支持直接分享链接访问

---

## 核心游戏循环

```
用户登录 → 查看城市地图 → 选择任务(或被系统指派) 
→ 用户自选专注时长 → 番茄钟开始 → 完成 
→ 资源写入城市 → 城市记录该用户贡献 → 返回地图看到变化
```

## 页面结构

### 1. 城市地图页（主页面）
- 全屏冰汽时代风格背景图（AI生成，固定图片）
- 城市当前划分为5个可点击区块，未来持续扩展：
  - ⛏️ **资源区**（煤矿/伐木场/收集小屋）
  - 🏠 **居住区**（帐篷/工人宿舍）
  - 🏥 **医疗区**（医疗室）
  - 🍲 **食物区**（猎人小屋/伙食食堂）
  - 🔭 **探索区**（哨站/工作站）
- 每个区块有SVG热区覆盖在背景图上
- **Hover效果**：气泡显示区块名称、当前状态、正在此处工作的人数
- **点击效果**：弹出区块详情面板
- 顶部状态栏：显示城市当前资源总量（煤/木材/钢材/食物/药品）、人口

### 2. 区块详情面板（Modal）
- 区块名称、描述、当前状态
- 正在此处工作的人数（如：「当前有3人正在工作」）
- 可用任务列表（每个任务显示：任务名称、预计产出或效果、时长由用户选择）
- **"前往工作"按钮** → 进入专注页面
- 若区块内建筑由某用户建造：显示「此建筑由[用户名]建造」

### 3. 专注页面
- 冰汽时代风格插画（根据区块类型切换，共5张）
- 当前任务名称（无额外描述文字）
- 用户自选专注时长（25分钟 / 45分钟 / 60分钟，或自定义）
- 番茄钟倒计时大字显示
- 背景音控制：根据区块类型自动播放对应背景音效
  - 资源区：挖掘声/斧声+风声
  - 居住区：炉火声+低语声
  - 医疗区：安静+微风
  - 食物区：篝火声+自然声
  - 探索区：雪地脚步声+风声
- 音乐播放器（lofi风格，可切换曲目）
- **放弃按钮**（二次确认后返回地图，不计入贡献）

### 4. 任务完成页
- 动画：资源入库效果
- 文字叙事：「{用户名}完成了{任务名}，为新伦敦贡献了{资源描述}」
- 若触发建造完成：「{建筑名}已建成，它将被命名为{用户名}建造的{建筑名}」
- 返回城市按钮

### 5. 用户个人页
- 用户名
- 累计专注时长
- 参与建造的建筑列表
- 历史贡献记录时间线

## 任务系统

### 任务列表（当前版本，未来持续扩展）

**资源区**
- 采集煤炭
- 采集木材
- 采集钢材
- 建造收集小屋（建造类，完成后建筑以用户命名）
- 参与固定收集工作

**居住区**
- 建造帐篷（建造类，完成后建筑以用户命名）

**医疗区**
- 建造医疗室（建造类，完成后建筑以用户命名）
- 医疗室值班

**食物区**
- 建造猎人小屋（建造类，完成后建筑以用户命名）
- 狩猎
- 建造伙食食堂（建造类，完成后建筑以用户命名）
- 食堂工作

**探索区**
- 建造工作站（建造类，完成后建筑以用户命名）
- 工作站研究
- 建造哨站（建造类，完成后建筑以用户命名）
- 哨站探索

### 任务分类规则
- **建造类**：消耗城市资源，完成后产出建筑，建筑永久以该用户命名
- **采集类**：产出资源，写入城市资源库，记录贡献数量
- **工作类**：产出服务性效果（士气/药品/食物），记录贡献次数

### 系统自动指派逻辑
当用户选择「被指派」时，后端根据以下优先级分配：
1. 城市当前最缺乏的资源 → 对应采集任务
2. 资源充足时 → 优先推送建造类任务
3. 建造类任务资源不足时 → 回退采集任务

## 多人共享机制
- 无需实时WebSocket，使用轮询（每30秒刷新城市状态）
- 所有用户共享同一个城市数据，贡献实时累加
- 「当前X人正在工作」通过查询进行中的专注session得出

---

## 城市日志系统
- 全局城市日志，所有人可见
- 每次专注完成写入一条日志
- 格式：`[时间] {用户名} {贡献描述}`
- 示例：`[今天 14:32] 王五 在资源区采集了30单位煤炭`
- 示例：`[今天 09:15] 小明 建造了帐篷，它被命名为「小明的帐篷」`

---

## 视觉风格要求
- 色调：深蓝、暗灰、暖橙（炉火色）
- 字体：衬线体，带有工业/蒸汽朋克气质
- 背景图：AI生成，冰天雪地中的蒸汽朋克工业城市，固定不动
- 区块热区：半透明橙色描边，hover时高亮
- 整体氛围：压抑但有希望，寒冷但有炉火

---

## Demo范围（MVP）

**包含:**
- 用户注册/登录（用户名即可）
- 城市地图 + 5个区块 + hover/点击交互
- 任务系统（含上述全部任务）
- 用户自选专注时长 + 番茄钟 + 背景音
- 完成后资源/效果写入城市
- 城市日志 + 建筑命名
- 用户贡献记录页

**不包含（后续版本）:**
- 决策事件系统
- 城市视觉成长变化（区块图片随发展切换）
- 区块历史贡献记录面板
- 移动端适配
- 社交功能
